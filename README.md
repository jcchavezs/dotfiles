# dotfiles

## Prerequisites

- [SSH key added into github account](https://help.github.com/en/articles/adding-a-new-ssh-key-to-your-github-account)
- git
- iterm
- vscode

## Install

```
cd ~
git clone https://github.com/jcchavezs/dotfiles .dotfiles
./.dotfiles/install.sh
```

## Backup
```
~/.dotfiles/backup.sh
```
